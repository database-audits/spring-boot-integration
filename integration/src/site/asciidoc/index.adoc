= Database Audits — Spring Boot Integration

`database-audits-spring-boot-integration` wires the link:https://database-audits.github.io/core[database-audits-core]
audit library into a Spring Boot test context. One `@Import` enables ten database audits across three families:
catalog (schema metadata), JPA (entity/schema agreement), and runtime (live SQL execution).

== Dependency

[source,xml]
----
<dependency>
  <groupId>io.github.database-audits</groupId>
  <artifactId>database-audits-spring-boot-integration</artifactId>
  <version>VERSION</version>
  <scope>test</scope>
</dependency>
----

*Note:* Replace `VERSION` with the latest released version.

See link:usage.html[Usage] to enable the audits and write your first tests, link:audits.html[Audits] for what
each audit checks, link:exclusions.html[Exclusions] to suppress known violations, and
link:archetype.html[Archetype] to scaffold a ready-to-run example suite into your project.
